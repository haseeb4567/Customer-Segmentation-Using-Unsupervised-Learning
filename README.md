📌 Project Overview

This project focuses on Customer Segmentation using Unsupervised Machine Learning techniques. The goal is to group customers into meaningful clusters based on their behavioral and demographic patterns without using labeled data. These insights can help businesses improve targeted marketing, customer retention, and personalized recommendations.

🎯 Objectives
Perform exploratory data analysis (EDA) on customer data
Preprocess and clean the dataset for modeling
Apply unsupervised learning algorithms (e.g., K-Means, Hierarchical Clustering)
Determine optimal number of clusters using methods like Elbow Method and Silhouette Score
Visualize customer segments for better interpretation
Derive actionable business insights from clusters
📊 Techniques Used
Data Preprocessing (handling missing values, scaling, encoding)
Exploratory Data Analysis (EDA)
K-Means Clustering
Hierarchical Clustering
Principal Component Analysis (PCA) for dimensionality reduction
Cluster Evaluation Metrics (Silhouette Score, Elbow Method)
🧠 Key Insights
Customers were successfully grouped into distinct segments based on behavior patterns
Each cluster represents a unique customer profile (e.g., high-value customers, frequent buyers, low-engagement users)
Businesses can use these segments for targeted marketing and personalized strategies
